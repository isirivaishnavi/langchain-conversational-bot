\# Conversational Knowledge Bot using LangChain



\## Overview

This project demonstrates a conversational AI bot built using LangChain.

The bot supports memory and can hold context-aware conversations.



\## Features

\- LangChain Conversation Chain

\- Conversational Memory

\- OpenAI LLM Integration



\## Setup Instructions

1\. Clone the repository

2\. Install dependencies:

&nbsp;  pip install -r requirements.txt

3\. Add your OpenAI API key in .env file

4\. Run:

&nbsp;  python main.py
